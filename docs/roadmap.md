# Roadmap

- Automatically remove nodes in NetBox when removed in Proxmox (aligned with VM cleanup behavior)
- Add per-object sync actions for individual nodes/VMs
- Improve periodic sync scheduling through NetBox jobs
- Improve documentation and test coverage for sync mapping
- Extend multi-cluster support
